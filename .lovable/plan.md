
# Zaki's Portfolio – Single-Page React App

## Overview
Convert your 4 standalone HTML portfolio pages into a unified React application with smooth navigation between sections.

## Pages & Routing

### 1. Home Page (`/`)
- Dark-themed header with your name, subtitle ("Year 12 Student"), and intro text
- Animated particle background
- **Projects grid** with 3 clickable cards (REBOUND, Project Euler, LeetCode) that link to their dedicated pages
- **Skills section** with cards for Problem Solving, Astrophysics, and Jiu-Jitsu
- Footer

### 2. REBOUND Astrophysics Page (`/rebound`)
- Space-themed dark design with starry/shooting star background
- Header with title and logo area
- Experience description section
- Code examples and simulation details
- Two-column content layout
- Back to home navigation

### 3. Project Euler Page (`/euler`)
- Warm gold/amber themed design
- Progress bar with animated counter (101 solved / 971 total)
- About section describing your experience
- Links to Project Euler website and back to portfolio

### 4. LeetCode Page (`/leetcode`)
- Dark theme with orange LeetCode accent colors
- Difficulty breakdown cards (Easy, Medium, Hard) with stats
- Experience description card
- Animated counters and progress indicators
- Back to home navigation

## Design
- Each page preserves its unique color scheme and visual identity from the original HTML files
- Smooth page transitions and animations (fade-ins, slide-ups, counters)
- Fully responsive for mobile and desktop
- Font Awesome icons throughout
- Animated backgrounds (particles on home, stars on REBOUND page)

## Navigation
- Project cards on the home page link to individual project pages
- Each project page has a "Back to Portfolio" link
- Simple, clean navigation flow
